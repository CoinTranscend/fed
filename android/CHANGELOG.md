# Fed Dashboard — Changelog

## v1.9 (2026-03-16)
- **Home screen widget** — shows LIIMSI, LLMSI, ISI, LSI, RRI with score, month-over-month delta, regime label, and 6-month sparkline
- Widget uses a light whitish-blue background with dark navy text
- Manual-refresh only (tap ↻) — no background auto-polling, zero battery impact when idle
- Sparklines rendered at high resolution (280×56) with thin 1.2px anti-aliased lines for crisp display
- Widget settings in Settings page: enable/disable toggle + "Refresh Now" button + active instance count
- Next scheduled economic release shown in widget footer
